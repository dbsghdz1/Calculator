# 📱Calculator-Project
<br>

## 💻 프로젝트 소개
<br>
이 프로젝트는 Swift언어를 사용하여 Xcode에서 개발한 애플리케이션입니다.<br>
StoryBoard를 이용한 방법과 Code를 이용해 만든 두가지 파일이 있습니다.<br>
사용자는 아이폰 버튼들을 이용해 실제 계산기와 비슷하게 계산을 해볼수 있습니다.<br><br>
